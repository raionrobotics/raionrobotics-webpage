Raisin External API v1.0.0
==========================

외부 시스템에서 Raibo2의 자율주행 기능을 제어하기 위한 API 문서입니다.

`English Version <../en/index.html>`_

System Requirements
-------------------

- **OS**: Ubuntu 22.04 이상
- **Architecture**: x86_64, arm64
- **Dependencies**: Eigen3, OpenSSL, PCL
