Quick Start
===========

This guide will help you get up and running with Raisin Master quickly.

Prerequisites
-------------

Before you begin, ensure you have the following installed on your system:

*   **Operating System**: Linux (Ubuntu 20.04+ recommended)
*   **Python**: 3.8 or higher
*   **C++ Compiler**: GCC 9+ or Clang 10+
*   **Build Tools**:
    *   CMake (3.16+)
    *   Ninja
*   **Version Control**: Git

Installation
------------

1.  **Clone the repository**:

    .. code-block:: bash

        git clone https://github.com/raionrobotics/raisin_master.git
        cd raisin_master

2.  **Install Python dependencies**:

    .. code-block:: bash

        pip install -r requirements.txt

3.  **Create configuration**:

    .. code-block:: bash

        cp configuration_setting_example.yaml configuration_setting.yaml

Basic Workflow
--------------

1.  **Setup the workspace**:

    This command scans your ``src/`` directory, generates headers, and configures the build system.

    .. code-block:: bash

        ./raisin.sh setup

    .. note::
        You can also call Python directly: ``python3 raisin.py setup``.

2.  **Build the project**:

    Compile all packages in your workspace.

    .. code-block:: bash

        ./raisin.sh build --type release

3.  **Source the environment** (if applicable):

    After building, you may need to source the setup script to add built binaries to your path.

    .. code-block:: bash

        source install/setup.bash  # (If generated)
